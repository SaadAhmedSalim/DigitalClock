# DigitalClock

In this repository, I am going to build a beautiful GUI based digital clock using the python Tkinter module.

<img src=https://github.com/SaadAhmedSalim/DigitalClock/blob/master/Digital_Clock/clock.png>

## Requirements 
    
    Python
    
    tkinter
    
    Pycharm
